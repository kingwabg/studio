## 실측 일치 (건강 — 회귀 의심 때만 재확인)

셀 선택(클릭·드래그 승격·Shift+클릭·Ctrl+클릭 비연속 — 한컴에 없는 우리 확장·F5 3단계) ·
셀 Ctrl+A 3단계 확장(2026-07-12 수정 ✅) · Tab/Shift+Tab(마지막 셀 Tab=행 자동 추가) ·
방향키 셀 경계 이동 · Home/End · 셀 병합/나누기(다중 범위·균등 높이) · 행열 삽입(4방향
1~63) · F5 범위 Delete 일괄 삭제 · 셀 서식(배경·테두리·안쪽 여백·세로 정렬·글 방향) ·
리사이즈 3종(hover 핸들·내부 경계 드래그·키보드) · 헤더행 반복 · 캡션 9방향 · 대각선 ·
계산식 22종 · 천단위/자릿점 · 표 객체 복사/붙여넣기 · 전치(transpose) 복사/붙여넣기.

**애매(TS에서 검증 불가)**: 엑셀/외부 표 HTML 붙여넣기 — pasteHtml이 wasm 내부로 위임,
실측으로만 확인 가능 (필요 시 실붙여넣기 시나리오로 검증).

## 갭

| # | 기능 | 우리 상태 (근거) | 기대 동작 | 점수 |
|---|---|---|---|---|
| G1 | 셀 범위(다중 셀) 복사 Ctrl+C | **죽은 경로** — onCopy가 텍스트 앵커만 검사(input-handler-keyboard.ts:1589), cellAnchor/cellFocus 무시 → F5/드래그 셀 선택 후 복사해도 무반응. 전치 복사만 생존 | 선택 셀들을 표 구조 그대로 복사·붙여넣기. 붙여넣기 그리드 의미론 = **미확정(③ 한컴 확인 필요)** | 3×3=**9 즉시** |
| G2 | 블록 합계/평균/곱 (Ctrl+Shift+S/A/P) | F5 선택 범위 무시 — 항상 `=FUNC(above)` 고정(table.ts:81) | 선택 블록 범위로 계산 | 2×3=**6 즉시** |
